📦 Microsoft Edge Portable

No installation required. Keeps all history, cookies, extensions, and settings even when moved to another device.


Latest Versions

  • Microsoft Edge Stable Portable 139.0.3405.102
  • Microsoft Edge Beta Portable 140.0.3485.14
  • Microsoft Edge Developer Portable 140.0.3485.6
  • Microsoft Edge Canary Portable 141.0.3500.0

Download the pre-built release, extract, and start using immediately.

🚀 Script Features

  • Downloads the latest Edge Stable, Beta, Dev, or Canary x64 from edge_installer_multi
  • Integrates Chrome++ for extra features
  • Manual update script preserves your settings and configurations

📁 Files and Their Purposes

  • chrome++.ini: configuration file for Chrome++
  • debloater.reg: removes unnecessary features from Microsoft Edge
  • default-apps-multi-profile.bat: sets the browser as the default application
  • update.bat: updates to the latest version

💻 How to Use

  1. Download the desired Edge channel version
  2. Extract the archive to your preferred location
  3. Run the executable file
  4. Enjoy your portable Microsoft Edge!

⚠️ Important Notice

⚠ Microsoft Defender Antivirus Warning

Due to the way Microsoft Edge is bypassed to run as a portable app, Microsoft Defender Antivirus may falsely flag it as a trojan.

Solution: Allow/whitelist the file and wait for Microsoft Defender's definitions to update and remove the false positive.
